Student Report Card

Overview

The Student Report Card is a Java console application designed to manage and evaluate student academic performance. The program accepts marks for multiple subjects, calculates the total marks, percentage, grade, and rank, and displays a detailed report card.

Features

- Input student details and marks
- Calculate total marks
- Calculate percentage
- Assign grades based on performance
- Generate a student report card
- Display student ranking

Technologies Used

- Java
- Scanner Class
- Arrays
- Conditional Statements
- Loops

Workflow

1. Enter student details.
2. Input marks for all subjects.
3. Calculate total marks obtained.
4. Calculate percentage.
5. Determine grade based on percentage.
6. Generate and display the report card.
7. Display rank information.

Concepts Practiced

- Variables and Data Types
- Arrays
- Loops
- Conditional Statements
- Methods
- Object-Oriented Programming Basics
- User Input Handling

Learning Outcomes

Through this project, I gained practical experience in processing student data, performing calculations, implementing grading systems, and developing a structured Java application using core programming concepts.
